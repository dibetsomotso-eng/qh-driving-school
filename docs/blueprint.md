# **App Name**: QH Driving School

## Core Features:

- Online Booking: Allow users to book driving lessons online.
- Service Listings: Display the QH Driving School's services such as driving lessons.
- Contact Form: Enable users to send inquiries through a contact form.
- WhatsApp Chat Button: Integrate a WhatsApp chat button for direct communication.
- Google Maps Integration: Embed Google Maps on the contact page for location details.
- Testimonial Display: Display a list of favorable student feedback.
- Newsletter Subscription: Allow users to subscribe to a newsletter.

## Style Guidelines:

- Primary color: Gold Yellow (#fcdd1e) to convey trust, clarity and optimism.
- Background color: Light Grey (#f2f0ee) to create a minimal and professional look. This color provides a muted backdrop that ensures readability and focuses attention on the content.
- Accent color: Black (#000000) to create contrast, ensuring key elements like headlines and call-to-action buttons stand out against the lighter background.
- Headline font: 'Playfair' serif with an elegant, fashionable, high-end feel; Body font: 'PT Sans' to balance style with legibility for a target audience of young professionals and students.
- Note: currently only Google Fonts are supported.
- Use professional icons that match the clean and structured website layout.
- Maintain a clean and minimal layout that builds trust and helps learners visualize their success. Adhere to a mobile-first responsive approach that works well across all devices.
- Implement subtle and purposeful animations such as fade-ins or hover effects to enhance user experience. Avoid gaudy, distracting animations.
